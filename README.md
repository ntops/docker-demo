# go-demo

```sh
go-demo # 项目根目录
    ├── docker-compose.yml  # compose 配置文件，用于容器编排
    └── helloworld          # 项目目录
        ├── log             # 项目日志文件存放目录
        ├── Dockerfile      # docker 配置文件，用于镜像构建
        ├── .dockerignore   # docker 配置文件，用于构建时忽略不必要文件
        ├── go.mod
        └── main.go
```
